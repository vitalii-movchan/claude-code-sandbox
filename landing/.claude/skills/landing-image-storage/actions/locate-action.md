# locate-action — контракт

Дать ссылку на изображение **или** показать список изображений. Общие правила
интерфейса — в [`action-list.md`](action-list.md). Реализации — секция
`## locate-action` в каждом файле `storages/<storage>.md`.

- **Вход:** `landing`, опц. `name` (если задан — локатор одного изображения;
  если опущен — список всех изображений лендинга в хранилище).
- **Выход (stdout):**
  - с `name` — один локатор (URL или путь);
  - без `name` — список: по строке на изображение, `имя` + `локатор`
    (+ размер, если хранилище его знает).
- **Инвариант:** для каждой строки списка локатор пригоден как вход
  `download-action`.
