# upload-action — контракт

Загрузить изображение в хранилище профиля. Общие правила интерфейса — в
[`action-list.md`](action-list.md). Реализации — секция `## upload-action` в
каждом файле `profiles/<profile>.md`.

- **Вход:** `landing`, `source` (путь к локальному файлу изображения),
  опц. `name` (имя/ключ в хранилище; по умолчанию — имя файла).
- **Выход (stdout):** канонический **локатор** загруженного изображения —
  то, что потом примет `locate-action`/`download-action` (для vercel это URL
  блоба, для local — путь в `pages/<landing>/img/`).
- **Инвариант:** после успешного upload изображение доступно через `locate-action`
  и скачиваемо через `download-action`.
