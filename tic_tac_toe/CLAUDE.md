# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Обзор

Коллекция самодостаточных одностраничных HTML-игр (крестики-нолики). Каждый `.html` — отдельная
законченная игра со встроенными HTML/CSS/JS, без зависимостей, сборки и тестов. Запуск — открытием
файла в браузере (`file://`), сервер не нужен.

## Файлы-игры

- `heaven.html` — светлая «небесная» тема (синее небо, облака), **намеренно слабый** бот.
- `dark-academia.html` — эстетика dark academia (сепия, золото, засечки), **непобедимый** бот.

## Архитектура

Все игры построены по одному шаблону (IIFE + `"use strict"`): человек играет `×` и ходит первым,
бот — `○`. Имена функций совпадают между файлами:

- `WIN_LINES` — 8 выигрышных линий как индексы 0–8; `board` — массив из 9 строк (`""`/`"×"`/`"○"`),
  UI-ячейки генерируются в цикле и хранятся в `cells`.
- `checkWinner(b)` → `{winner, line}` | `null`; `isFull(b)`; `emptyCells(b)`.
- `place(idx, player)` ставит символ и блокирует ячейку; `onCellClick` обрабатывает ход человека,
  затем через `setTimeout(~420мс)` делает ход бота.
- `checkEnd`, `highlight(line)`, `lockBoard`, `setStatus`, `restart` — поток конца партии и UI.

**Различие между играми — только в движке бота и в CSS.** Логика поля переиспользуется без изменений:
при новой версии копируй её, меняй движок бота (вызывается в `onCellClick`) и палитру/разметку.

### Движки ботов

- Слабый (`heaven.html`): `aiMove` — `findWinningMove` для выигрыша/блока с вероятностью (~70%),
  иначе `randomEmpty`.
- Непобедимый (`dark-academia.html`): `bestMove(b)` перебирает пустые клетки через
  `minimax(b, depth, isMaximizing)`. Оценка: победа `10 - depth`, поражение `depth - 10`, ничья `0`
  (быстрее выигрывать, дольше проигрывать). Поле 3×3 — оптимизация не нужна.

## Соглашения

- `<meta name="color-scheme">`: `only light` у `heaven.html`, `only dark` у `dark-academia.html` —
  чтобы Dark Reader и т.п. не инвертировали палитру. Сохраняй при правках оформления.
- Язык интерфейса и комментариев — русский.
- Палитра — через CSS-переменные в `:root`, не точечными правками.

## Проверка

Тестов нет, проверка ручная в браузере. Для непобедимого бота: сыграть несколько партий (должна
выходить только ничья или победа бота), проверить ловушку «угол → центр → противоположный угол»,
рестарт, подсветку выигрышной линии и отображение ничьей.
