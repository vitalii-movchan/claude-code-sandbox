# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working in this directory.

## Что это

`~/www/claude-code` — **песочница (sandbox)** для учебных и экспериментальных claude-проектов.
Не самостоятельный проект, а временный контейнер для других проектов.

Каждая подпапка (`tic_tac_toe/`, `landing/`, `research/` и т.п.) — отдельный изолированный проект
со своим `CLAUDE.md` и набором инструментов (скиллы, агенты, плагины).

## Правила песочницы

- **Корень держим чистым.** Единственный уместный факт на уровне корня — этот `CLAUDE.md`;
  информацию о конкретных проектах здесь не храним.
- **Проекты изолированы.** Каждый живёт в своей папке со своим `CLAUDE.md`, исполняется в
  своей среде, не делится состоянием с соседями.
- **Расширения — внутри проекта.** Скиллы, агенты и инструменты ставятся локально, видны только
  внутри своего проекта. Нужен новый — создавай его в самом проекте.
- **Готовый проект выносится наружу** — в отдельный git-репозиторий / пакет / плагин / скилл-агент.

## Как работать

Источник истины по проекту — его собственный `CLAUDE.md`. Работай внутри конкретного проекта,
соседние не трогай, если задача явно не про них.
